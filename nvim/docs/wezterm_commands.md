# WezTerm Cheat Sheet

## Tabs

| Action    | macOS                | Windows / Linux      |
| --------- | -------------------- | -------------------- |
| New Tab   | `CMD + T`            | `CTRL + SHIFT + T`   |
| Close Tab | `CMD + W`            | `CTRL + SHIFT + W`   |
| Next Tab  | `CTRL + TAB`         | `CTRL + TAB`         |
| Prev Tab  | `CTRL + SHIFT + TAB` | `CTRL + SHIFT + TAB` |

## Panes

| Action           | macOS             | Windows / Linux    |
| ---------------- | ----------------- | ------------------ |
| Split Vertical   | `CMD + SHIFT + %` | `CTRL + SHIFT + %` |
| Split Horizontal | `CMD + SHIFT + "` | `CTRL + SHIFT + "` |

## Pane Navigation

| Direction | macOS                | Windows / Linux         |
| --------- | -------------------- | ----------------------- |
| Move      | `CMD + OPT + Arrows` | `CTRL + SHIFT + Arrows` |

## Clipboard

| Action | macOS     | Windows / Linux    |
| ------ | --------- | ------------------ |
| Copy   | `CMD + C` | `CTRL + SHIFT + C` |
| Paste  | `CMD + V` | `CTRL + SHIFT + V` |

## Font / Zoom

| Action   | macOS     | Windows / Linux |
| -------- | --------- | --------------- |
| Increase | `CMD + =` | `CTRL + =`      |
| Decrease | `CMD + -` | `CTRL + -`      |
| Reset    | `CMD + 0` | `CTRL + 0`      |

## Window

| Action     | macOS         | Windows / Linux |
| ---------- | ------------- | --------------- |
| Fullscreen | `CMD + ENTER` | `ALT + ENTER`   |

## Utility

| Action        | macOS             | Windows / Linux    |
| ------------- | ----------------- | ------------------ |
| Reload Config | `CMD + SHIFT + R` | `CTRL + SHIFT + R` |
